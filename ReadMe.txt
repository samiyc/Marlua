#infos

l'ensemble des images, cartes, arrière-plan, fog, animations...
Quasiment tout est paramétrable dans des fichier .txt
Marlua.exe peut être lancé indépendamment des autres fichiers.

Les fichiers de configuration :
cfg - Paramétrage des tilesets (perso/monstre/déco), cartes jouables (menu) et l'arrière plan
entity_info - Paramétrage des images/animations à utiliser pour toutes les entités interactives du jeu

Les fichiers map :
green - Contient tout ce qui est au premier plan, ainsi que les entités
green_info - Paramétrage des caractères utilisés dans green pour le sol, les entités, etc...
green_background - Arrière-plan
green_background_info - Paramétrage de l'arrière-plan
green_far_away - Arrière-plan distant
green_far_away_info - Paramétrage de l'arrière-plan distant

Ce projet utilise le game engine LOVE 2D V11.1
https://love2d.org/#download
